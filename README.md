# LVC-and-Ada
This folder contains all the documents relative to my website http://lvcetada.e-monsite.com/

This folder contains three parts:
- AVR-Ada-Lib contains the librairies which come during the AVR\_Ada installation;
- Uno-Projects contains projects for Arduino Uno board;
- Mega-Projects contains projects for Arduino Mega board;
- Usefull-Librairies contains librairies for Arduino Uno, Arduino Mega and components.
